# Calipso AI Landing (cal-app-landing)

Static landing page for Calipso AI.

---

## Get the site live (2 steps)

1. **Create the repo on GitHub**  
   New repo named `cal-app-landing`, empty (no README, no .gitignore).

2. **Run the deploy script** (from this folder):

   ```bash
   cd /Users/admin/programming-exercises/_projects_/mob-apps/cal-app-landing
   chmod +x deploy.sh
   ./deploy.sh
   ```

   If your GitHub username isn’t `arturteplov`, run:
   ```bash
   ./deploy.sh https://github.com/YOUR_USERNAME/cal-app-landing.git
   ```

3. **Turn on GitHub Pages**  
   On GitHub: **Settings → Pages → Source**: “Deploy from a branch”.  
   Branch: **main**, Folder: **/ (root)** → Save.

4. **Open the site**  
   `https://arturteplov.github.io/cal-app-landing/` (or your username).
